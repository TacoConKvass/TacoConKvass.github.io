RL2.API

... is a modding API for Rogue Legacy 2. It allows for easeier manipulation of several parts of the game.

What can be maniuplated or added?

  • Enemies:
    • On spawn behaviour
    • Stats
    • Spawn conditions
    • Pre kill behaviour - allows death prevention
    • On kill behaviour
  • Player:
    • On spawn behaviour
    • Stats
    • Heir generation
    • Pre kill behaviour - allows death prevention
    • On kill behaviour
  • Abilities:
    • Stats
  • Map
    • Custom room icons
